# AI README Generator

AI-powered README generator that creates comprehensive project documentation from code analysis, using local LLMs (Ollama).

## Quick Start

```bash
pip install -e .
airg generate
```

## Features

- Analyzes project structure (files, imports, classes, functions)
- Generates complete README with installation, usage, and API docs
- Supports Python, JavaScript, Go, and Rust projects
- Multiple templates (standard, minimalist, detailed)
